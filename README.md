# AI Recommendation Project 🎯

## Project Overview

This project implements a **Content-Based Recommendation System** using **Python** and **Scikit-learn**. It recommends courses based on user interests by applying **TF-IDF Vectorization** and **Cosine Similarity** to measure how closely a user's input matches the available course descriptions.

## Features

- Takes user interests as input
- Uses TF-IDF Vectorization to process text
- Calculates similarity using Cosine Similarity
- Recommends the top matching courses
- Simple and interactive command-line interface

## Technologies Used

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity
